# Albatil Fitness v2 (React) — دليل النشر والتثبيت على الآيفون

## البنية
هذا الإصدار مبني بـ React (بدون خطوة Build — يشتغل مباشرة من الملفات عبر GitHub Pages):

```
v2/
  index.html
  App.jsx
  components/   (Charts.jsx, UI.jsx)
  pages/        (Home, Workouts, Library, ScheduleEditor, Session, Progress, Nutrition, Settings)
  hooks/        (useAppState.js)
  services/     (storage.js, firebase.js, ai.js)
  utils/        (data.js, helpers.js)
  manifest.webmanifest, sw.js, icon.svg
```

بياناتك القديمة (من النسخة الساكنة بجذر المستودع) تترحّل تلقائيًا أول مرة تفتح فيها v2 على نفس النطاق.

## الرفع على GitHub
ارفع مجلد `v2/` كامل إلى مستودعك (كمجلد فرعي، بجانب الملفات الحالية بجذر المستودع — لا تحذف شيء).

## فتحه كتطبيق على الآيفون
1. بعد رفع v2 وتفعيل GitHub Pages، افتح الرابط بهذا الشكل: `https://username.github.io/repo/v2/`
2. من Safari (لازم Safari تحديدًا، مو كروم): زر المشاركة (المربع والسهم) → **"إضافة إلى الشاشة الرئيسية"**
3. بيصير له أيقونة مستقلة على شاشتك تفتح بدون شريط المتصفح — يعمل حتى بدون إنترنت بعد أول فتح.

⚠️ **هذا تثبيت PWA على الشاشة الرئيسية — مو تطبيق حقيقي من App Store.** يعمل ويبدو كتطبيق تقريبًا، لكن نظام iOS يتعامل معه كصفحة ويب محفوظة، وليس تطبيقًا مُوقّعًا برمجيًا في الـ App Store. تطبيق App Store حقيقي مرتبط بـ Apple Health يتطلب تطوير Swift/Xcode كامل — خارج نطاق ما يقدر هذا المسار يسويه؛ لو تبي تنتقل لهذا لاحقًا نقدر نجهز حزمة تسليم للمطور (Claude Code) تبدأ من نفس منطق التطبيق.

## المزامنة السحابية (تفتح التطبيق من أي جهاز)
1. أنشئ مشروع مجاني على https://console.firebase.google.com
2. فعّل Authentication → Google، وفعّل Firestore Database
3. انسخ `firebase-config.example.js` باسم `firebase-config.js` وحط مفاتيح مشروعك فيه
4. ارفعه داخل مجلد `v2/`
5. من التطبيق → الإعدادات → المزامنة السحابية → سجّل دخول بجوجل → رفع/سحب بياناتك

بدون هذا الملف، التطبيق يشتغل عادي محليًا فقط (تصدير/استيراد يدوي يبقى شغال دائمًا).

## 🤖 التحليل الذكي
تبويب "التحليلات" بالتقدم فيه زر "احصل على ملاحظات ذكية" يحلل بياناتك (حجم التمرين، الأوزان، الأرقام القياسية، التغذية) ويطلع لك ملاحظات نصية بالعربية.

## الفرق عن الإصدار الساكن (جذر المستودع)
- React بدل JS خام + ملفات منفصلة (components/pages/hooks/services/utils) بدل ملف واحد ضخم
- رسوم بيانية أنعم (خطوط منحنية، تدرّج لوني، تمييز آخر نقطة)
- المزامنة السحابية مركزية أكثر (تسحب/تدفع تلقائيًا لما تكون مسجل دخول)
- كل الميزات السابقة موجودة: التمارين، التغذية، التقدم، التحليلات، الإنجازات، مكتبة التمارين، ملفات شخصية، نسخ احتياطي، PWA، وضع داكن/فاتح، تقارير، مؤقت راحة، تخصيص الجدول، حاسبة الدهون (البحرية الأمريكية)، حاسبة TDEE

النسخة الساكنة بجذر المستودع باقية شغالة زي ما هي — قارن بينهم وقولي أي وحدة تبيها تصير الأساسية.
