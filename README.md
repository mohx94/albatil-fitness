# Albatil Fitness V3 — Smart Coach

نسخة مجانية بالكامل وتعمل على GitHub Pages بدون خادم أو اشتراك.

## أبرز الإضافات
- مدرب ذكي محلي يحلل الالتزام والتغذية وتوازن العضلات.
- Body Score وتقرير كامل للجسم والأداء.
- تقرير متقدم بعد كل حصة وتقييم من 100.
- نظام XP ومستويات.
- خريطة توازن العضلات لآخر 14 يوم.
- اقتراح يومي بناءً على البيانات المسجلة.
- المحافظة على بيانات localStorage السابقة تلقائيًا.

## الرفع على GitHub
1. استبدل ملفات المستودع بمحتويات هذا المجلد.
2. ارفع الملفات إلى الفرع main.
3. افتح الموقع ثم حدّث الصفحة مرتين أو احذف نسخة الموقع القديمة من الكاش إذا لم يظهر التحديث فورًا.

## Firebase
الملف firebase-config.js موجود وفق إعداداتك الحالية. لا تضعه في مستودع عام إذا كنت لا تريد نشر إعدادات المشروع. مفاتيح Firebase Web ليست كلمة مرور، لكن يجب ضبط قواعد Firestore وقيود النطاقات بشكل صحيح.
