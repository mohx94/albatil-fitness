# Albatil Fitness v3.0

## طريقة التحديث على GitHub
انسخ الملفات التالية إلى جذر المستودع (استبدل الموجود):
- index.html
- manifest.webmanifest
- icon.svg
- sw.js

ثم من GitHub Desktop:
1. Commit message: `Albatil Fitness v3.0`
2. Commit to main → Push origin

بيانات المستخدمين الحالية (localStorage) تبقى محفوظة تلقائيًا وتترحّل بنفسها لأول مرة يفتح فيها التطبيق بعد التحديث — ما راح تنفقد.

## الجديد في v3.0

### 🏋️ التمارين
- 📚 مكتبة تمارين كاملة: نصيحة تنفيذ + بدائل مقترحة لكل تمرين، متاحة من شاشة "التمارين"
- 🎯 نظام تدريجي ذكي (auto progression): يقترح لك وزن الجولة الأولى تلقائيًا بناءً على آخر مرة تدربت فيها هالتمرين
- ℹ️ زر "التفاصيل والبدائل" داخل كل تمرين أثناء الحصة

### 📊 التحليلات
- تبويب جديد "التحليلات" داخل التقدم: ملخص آخر 7 أيام وآخر 30 يوم (عدد الحصص، الحجم الكلي، المتوسط)
- رسم بياني لحجم التمرين حسب كل عضلة خلال آخر أسبوع

### 🍗 التغذية
- 🧮 حاسبة سعرات تلقائية (TDEE) من الطول والعمر والجنس ومستوى النشاط والهدف — تقترح لك أهداف السعرات والبروتين والكارب والدهون
- ⚡ وجبات سريعة جاهزة تُضاف بضغطة واحدة
- 🔍 بحث بالباركود (تجريبي): اربط باركود بأي صنف تضيفه بنفسك ثم دوّر عليه بالرقم

### 👥 حسابات متعددة
- ملفات شخصية متعددة على نفس الجهاز (بيانات، تمارين، تغذية منفصلة لكل ملف) من شاشة الإعدادات

### 🔔 PWA
- تفعيل إشعار عند انتهاء مؤقت الراحة (حتى لو التطبيق بالخلفية)
- تخزين أوفلاين محسّن (تحديث تلقائي بالخلفية + نسخة احتياطية عند انقطاع الشبكة)

### ☁️ المزامنة السحابية
- مو متوفرة بعد بهذا الإصدار — حاليًا استخدم زر "تصدير/استيراد البيانات" بالإعدادات لنقل بياناتك بين الأجهزة يدويًا. قريبًا بنضيف مزامنة تلقائية.

## ملاحظة
قاعدة الأكل والتمارين تقديرية، تقدر تعدّلها أو تضيف عليها من داخل التطبيق نفسه.
